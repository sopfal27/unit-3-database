About CineMatch

Our Mission

CineMatch helps movie lovers discover their next favorite film through personalized recommendations and a curated collection of cinema's finest works.

Built With Modern Tech

CineMatch is a student project built with:

  • Flask - Python web framework
  • Bootstrap 5 - Responsive design
  • SQLAlchemy - Database management (coming in Lesson 3.1!)
  • AI Integration - Personalized recommendations (coming later!)

Learning Journey

This application is part of Unit 3: Database Foundation in our CTE Web Development course. Throughout this unit, we'll learn:

  • Database design and SQLAlchemy ORM
  • CRUD operations (Create, Read, Update, Delete)
  • Forms and user input validation
  • Search and filtering functionality
  • Pagination for large datasets
